Linear → Clockify Timer

Chrome extension — Clockify time-tracker Linear és HelpScout integrációval.

⬇ Letöltés (linear-clockify.zip) Mindig a legfrissebb verzió. Korábbi release-ek.
Miért kell zipből telepíteni? Az extension még nincs fent a Chrome Web Store-on (review alatt), addig kézzel kell telepíteni. Egyszeri ~5 perc.

Telepítés röviden

  1. Töltsd le a zipet (fenti gomb)
  2. Csomagold ki egy állandó mappába (pl. ~/Applications/linear-clockify/) — ne mozgasd utána!
  3. Nyisd meg: chrome://extensions/
  4. Jobb felül: Developer mode ON
  5. Bal felül: Load unpacked → válaszd ki a kicsomagolt mappát
  6. Tűzd ki a toolbarra (🧩 puzzle ikon → pin)
  7. Jobb klikk az extension ikonon → Options → Clockify + Linear API key beillesztése

→ Részletes, képernyőképes útmutató

Frissítés

Ha új verzió jön ki, töltsd le a zipet újra, csomagold ki ugyanabba a mappába (felülírva), majd a chrome://extensions/ oldalon nyomj a 🔄 Reload gombra a kártyán.
