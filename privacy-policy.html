Linear → Clockify Timer — Privacy Policy

Last updated: 2026-04-16

What this extension does

This Chrome extension adds a Clockify time tracking button to Linear issue pages. It lets you start, stop, and log time entries directly from Linear.

Data collected and stored locally

The following data is stored in your browser's local storage (chrome.storage.local) and never leaves your device except as described below:

  • Clockify API key — used to authenticate with the Clockify API
  • Linear API key — used to fetch issue details (title, project, parent issue) from the Linear API
  • Clockify Workspace ID — identifies which Clockify workspace to use
  • Team → Project mapping — maps Linear teams to Clockify project names
  • User preferences — auto-stop setting

External services

The extension communicates with two external APIs, using the API keys you provide:

  • Clockify API (api.clockify.me) — to start/stop timers, create time entries, and look up projects
  • Linear API (api.linear.app) — to fetch issue details (title, project name, parent issue identifier)

No data is sent to any other server, analytics service, or third party.

Data NOT collected

  • No personal information (name, email, etc.)
  • No browsing history or page content beyond Linear issue identifiers
  • No analytics, telemetry, or tracking of any kind
  • No cookies

Data sharing

This extension does not sell, share, or transfer any data to third parties. Data only flows between your browser and the Clockify/Linear APIs that you explicitly configured.

Permissions explained

  • storage — save your settings and API keys locally
  • alarms — periodically update the elapsed time display on the browser action badge
  • tabs — detect when Linear tabs are closed (for the auto-stop feature)
  • host_permissions — access Clockify API, Linear API, and inject the timer button on Linear pages
